⚡ Como o Fluxo Funciona
01
O DHT22 mede temperatura e umidade a cada 2 segundos e envia via GPIO ao ESP32
02
O ESP32 processa os dados e publica via Wi-Fi no broker MQTT usando protocolo seguro TLS
03
O Node-RED recebe os dados, salva no MySQL, envia alertas via WhatsApp e aciona o CLP
04
O CLP WEG PLC300 executa a lógica ladder: se T ≥ 40°C → aciona saídas digitais para os atuadores
05
Os atuadores (AC, sirene, fechadura, válvula) são acionados automaticamente para proteger os moradores
